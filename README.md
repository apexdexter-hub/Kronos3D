# Kronos3D

Independent 3D Modeling engine for Android ARM64. Inspired by Blender mesh operations and Blockbench workflows.

## Stack
- C++ Android NDK (OpenGL ES 3.2 via MobileGlues translation layer)
- ImGui (User Interface)
- GLM (Graphics Mathematics)
- Assimp (File formats importer/exporter)
